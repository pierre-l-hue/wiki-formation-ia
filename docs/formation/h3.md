# H3 — Structurer un prompt efficace

*Méthodes ROFT, CROFT, SOCRATE, FAI et Reverse Prompting pour maîtriser l'ingénierie de prompt.*

---

## Objectifs de cette séance

- **Maîtriser les 4 grandes méthodes de structuration** : ROFT, CROFT, SOCRATE et FAI.
- **Choisir la bonne grille** selon l'urgence, la complexité et l'enjeu stratégique.
- **Générer et itérer ses prompts** en direct à l'aide de l'outil interactif.
- **Préparer l'Exercice 2 de certification** : Conception d'un prompt d'article de blog + prompt pour visuel IA.

---

## Rappel : La grille ROFT

| Lettre | Description | Mauvaise pratique | Bonne pratique |
| :---: | :--- | :--- | :--- |
| **R** | **Rôle** | *"Aide-moi à rédiger."* | *"Tu es un expert en communication B2B..."* |
| **O** | **Objectif** | *"Écris-moi un mail."* | *"Rédige un email de relance pour un devis sans réponse..."* |
| **F** | **Format** | *"Donne-moi des idées."* | *"Présente le mail sous forme de 3 paragraphes courts (< 150 mots)..."* |
| **T** | **Ton** | *"Fais un truc pro."* | *"Adopte un ton empathique, professionnel et assertif, sans jargon."* |

---

## Les déclinaisons avancées

### 1. CROFT — La version augmentée (ROFT + Contexte)
En ajoutant 2 à 3 phrases de **Contexte**, vous évitez les réponses génériques ("passer-partout") et forcez l'IA à s'adapter immédiatement à votre environnement professionnel.

!!! example "Exemple CROFT"
    > *"**Contexte :** Je suis responsable RH dans une PME de 30 salariés en croissance. Nous n'avons pas encore de charte de télétravail officielle.<br>**Rôle :** Tu es juriste en droit du travail.<br>**Objectif :** Rédige une note d'information claire sur la mise en place du télétravail.<br>**Format :** Une note interne structurée avec 4 rubriques claires.<br>**Ton :** Pédagogique, rassurant et rigoureux."*

### 2. SOCRATE — Pour les cas complexes ou stratégiques
Utilisez SOCRATE pour les sujets délicats (gestion de crise, négociation, dossiers volumineux) :

| Lettre | Élément | Question d'alignement |
| :---: | :--- | :--- |
| **S** | **Situation** | Quel est le problème initial ou le point de départ ? |
| **O** | **Objectif** | Quel est le résultat précis attendu ? |
| **C** | **Contexte** | Quelles sont les contraintes à respecter (temps, budget, cible) ? |
| **R** | **Rôle** | Quelle posture ou quel niveau d'expertise doit adopter l'IA ? |
| **A** | **Attentes** | Quels sont les critères de réussite ou limites d'acceptation ? |
| **T** | **Ton** | Quel registre de langue ou style adopter ? |
| **E** | **Exemples** | Quels exemples ou modèles types fournir à l'IA pour l'isoler ? |

### 3. FAI — Pour les besoins ultra-rapides (< 2 minutes)
- **F**ormule : *"Rédige une accroche..."*
- **A**ction : *"Propose 3 variantes..."*
- **I**ntention : *"Pour captiver l'attention sur LinkedIn."*

---

## Quelle méthode choisir ?

| Contrainte de temps | Complexité du sujet | Méthode recommandée | Cas d'usage type |
| :---: | :---: | :---: | :--- |
| ⚡ < 2 min | Faible | **FAI** | Accroche LinkedIn, correction de texte, idée rapide |
| ⏱️ 5 min | Moyenne | **ROFT** | Email classique, compte-rendu, résumé |
| 🎯 10 min | Élevée | **CROFT** | Post de blog, proposition commerciale, note interne |
| 🔍 > 15 min | Critique | **SOCRATE** | Gestion de conflit, plan stratégique, cas RH complexe |

(function() {
  const roleSelect = document.getElementById('prompt-role');

  const tonSelect = document.getElementById('prompt-ton');
  const tonCustom = document.getElementById('prompt-ton-custom');

  let showSocrate = false;

  function updatePreview() {
    const valRole = getValue(roleSelect, roleCustom);
    const valFormat = getValue(formatSelect, formatCustom);
    const valTon = getValue(tonSelect, tonCustom);

    let method = "ROFT";
    let text = `Rôle : Tu es un ${valRole}.\nObjectif : ${objectif.value}.\nFormat : ${valFormat}.\nTon : ${valTon}.`;

    if (showContexte || showSocrate) {
      text = `Contexte : ${contexte.value}\n` + text;
      method = "CROFT";
    }

    if (showSocrate) {
      text += `\nAttentes particulières : ${attentes.value}`;
      method = "SOCRATE";
    }

    preview.textContent = text;
    methodTag.textContent = method;
  }

  [
    { select: roleSelect, custom: roleCustom },
    { select: formatSelect, custom: formatCustom },
    { select: tonSelect, custom: tonCustom }
  ].forEach(pair => {
    pair.select.addEventListener('change', () => {
      handleCustomToggle(pair.select, pair.custom);
      updatePreview();
    });
    pair.custom.addEventListener('input', updatePreview);
  });

  btnCroft.addEventListener('click', () => {
    showContexte = !showContexte;
    fieldContexte.style.display = showContexte ? 'block' : 'none';
    btnCroft.style.background = showContexte ? '#1a5fb4' : 'transparent';

    updatePreview();
  });

  btnSocrate.addEventListener('click', () => {
    showSocrate = !showSocrate;
    fieldSocrate.style.display = showSocrate ? 'block' : 'none';
    if (showSocrate && !showContexte) {
      showContexte = true;
      fieldContexte.style.display = 'block';
      btnCroft.style.background = '#1a5fb4';
      btnCroft.style.color = '#fff';
    }
    btnSocrate.style.background = showSocrate ? '#1a5fb4' : 'transparent';
    btnSocrate.style.color = showSocrate ? '#fff' : 'var(--md-typeset-color, #000)';
    updatePreview();
  });

    });

## Techniques d'optimisation avancées

### 1. Demander à l'IA d'interroger avant de répondre
Pour éviter les suppositions, ajoutez cette consigne simple à la fin de vos prompts :
> *"Avant de générer la réponse, pose-moi 3 questions d'éclaircissement si certains points te semblent flous."*

### 2. Le Reverse Prompting (Rétro-ingénierie)
Si vous observez un résultat remarquable (un excellent article, une image bluffante ou un e-mail parfait), vous pouvez demander à l'IA d'en déduire le prompt d'origine :

!!! tip "Prompt type pour Reverse Prompting"
    > *"Analyse le texte/l'image ci-dessous. Reconstitue le prompt détaillé (Rôle, Contexte, Objectif, Format, Ton) qui a permis d'obtenir ce résultat exact, afin que je puisse le réutiliser pour un autre sujet."*
