## Rappel de ROFT

| Lettre | Description | Mauvais | Bon |
|---|---|---|---|
| **R** — Rôle | Donner un rôle clair | « Aide-moi » | « Tu es un expert en communication » |
| **O** — Objectif | Ce que l'on veut exactement | « Écris-moi un mail » | « Rédige un mail à mon client principal pour le remercier de sa commande » |
| **F** — Format | Structure, longueur | « Donne-moi des idées » | « Le mail sera en 2 parties : remerciement (2-3 phrases) puis relance (1 phrase) » |
| **T** — Ton | Style, énergie, ce qu'il faut éviter | « Fais un truc professionnel » | « Chaleureux et rassurant, vouvoiement, pas de jargon » |

## CROFT — la version complète (ROFT + Contexte)

Ajouter le **Contexte** en 2-3 phrases : qui êtes-vous, ce que vous faites, à qui vous vous adressez. Sans contexte, la réponse est générique ; avec contexte, elle est directement utilisable.

!!! example "Exemple CROFT"
    *Je suis client chez Free. Je souhaite résilier mon abonnement box sans engagement au 1er février. Je veux que le service client me réponde. Agis comme un juriste. Fais-moi une lettre de résiliation FAI qui inclue des éléments juridiques en 2 parties.*

## SOCRATE — quand c'est complexe

| Lettre | Question |
|---|---|
| **S** — Situation | Quel est le problème ? |
| **O** — Objectif | Quel résultat je veux de l'IA ? |
| **C** — Contexte | Quels détails importants ? |
| **R** — Rôle | Quel rôle doit avoir l'IA ? |
| **A** — Attentes | Quel format/structure ? |
| **T** — Ton | Quel style/langage ? |
| **E** — Exemples | Donne-moi 2-3 exemples |

!!! example "Exemple SOCRATE"
    Un fournisseur est en retard de paiement sur une commande déjà livrée, et ce retard crée des tensions sur la trésorerie. « Je suis responsable logistique dans une PME de 20 salariés... Agis comme un responsable achats expérimenté. L'email doit comporter un objet clair et 3 paragraphes (~150 mots)... Ton courtois mais ferme. Propose 2 variantes. »

## FAI — quand la rapidité prime

| Lettre | Question |
|---|---|
| **F** — Formule | Quelle phrase simple résume la demande ? |
| **A** — Action | Que doit faire l'IA concrètement ? |
| **I** — Intention | À quoi va servir le résultat ? |

## Quelle méthode pour quel besoin ?

| Contexte | Méthode | Usage |
|---|---|---|
| En 2 min ? | **FAI** | Brainstorming, email rapide, relance courte |
| Entre 5 et 10 min ? | **ROFT** | Script d'appel, contenu ou email classique |
| Pour être personnalisé ? | **CROFT** | Mail important, comptes rendus |
| En cas de blocage ? | **SOCRATE** | Retard paiement, relance, besoin de creuser |

!!! tip "Bonus"
    Ne pas hésiter à combiner les méthodes. Ex : ROFT pour le cadre + SOCRATE pour creuser une objection = le meilleur des deux mondes.

<script>
(function() {
  const roleSelect = document.getElementById('prompt-role');
  const roleCustom = document.getElementById('prompt-role-custom');
  
  const formatSelect = document.getElementById('prompt-format');
  const formatCustom = document.getElementById('prompt-format-custom');
  
  const tonSelect = document.getElementById('prompt-ton');
  const tonCustom = document.getElementById('prompt-ton-custom');

  const objectif = document.getElementById('prompt-objectif');
  const contexte = document.getElementById('prompt-contexte');
  const attentes = document.getElementById('prompt-attentes');
  const exemples = document.getElementById('prompt-exemples');
  
  const preview = document.getElementById('prompt-preview');
  const methodTag = document.getElementById('prompt-method-tag');
  
  const btnCroft = document.getElementById('btn-croft');
  const btnSocrate = document.getElementById('btn-socrate');
  const fieldContexte = document.getElementById('field-contexte');
  const fieldSocrate = document.getElementById('field-socrate');

  if (!roleSelect || !preview) return;

  let showContexte = false;
  let showSocrate = false;

  function getValue(selectEl, customEl) {
    return selectEl.value === 'custom' ? (customEl.value || '...') : selectEl.value;
  }

  function handleCustomToggle(selectEl, customEl) {
    if (selectEl.value === 'custom') {
      customEl.style.display = 'block';
      customEl.focus();
    } else {
      customEl.style.display = 'none';
    }
  }

  function setBtnActive(btn, isActive) {
    if (isActive) {
      btn.style.background = "#1a5fb4";
      btn.style.color = "#ffffff";
      btn.style.borderColor = "#1a5fb4";
    } else {
      btn.style.background = "#ffffff";
      btn.style.color = "#1a1a1a";
      btn.style.borderColor = "#999999";
    }
  }

  function updatePreview() {
    const valRole = getValue(roleSelect, roleCustom);
    const valFormat = getValue(formatSelect, formatCustom);
    const valTon = getValue(tonSelect, tonCustom);

    let method = "ROFT";
    let text = "Tu es " + valRole + ". " + objectif.value + ". Format : " + valFormat + ". Ton : " + valTon + ".";

    if (showContexte || showSocrate) {
      text = "Contexte : " + contexte.value + ". " + text;
      method = "CROFT";
    }
    if (showSocrate) {
      text += " Attentes : " + attentes.value + ". Exemples : " + exemples.value + ".";
      method = "SOCRATE";
    }
    preview.textContent = text;
    methodTag.textContent = "Méthode utilisée : " + method;
  }

  // Ecouteurs sur les sélecteurs avec option personnalisée
  [
    { select: roleSelect, custom: roleCustom },
    { select: formatSelect, custom: formatCustom },
    { select: tonSelect, custom: tonCustom }
  ].forEach(pair => {
    pair.select.addEventListener('change', () => {
      handleCustomToggle(pair.select, pair.custom);
      updatePreview();
    });
    pair.custom.addEventListener('input', updatePreview);
  });

  // Ecouteurs sur les boutons
  btnCroft.addEventListener('click', () => {
    showContexte = !showContexte;
    setBtnActive(btnCroft, showContexte);
    fieldContexte.style.display = showContexte ? 'block' : 'none';
    updatePreview();
  });

  btnSocrate.addEventListener('click', () => {
    showSocrate = !showSocrate;
    setBtnActive(btnSocrate, showSocrate);
    fieldSocrate.style.display = showSocrate ? 'block' : 'none';
    if (showSocrate) {
      showContexte = true;
      setBtnActive(btnCroft, true);
      fieldContexte.style.display = 'block';
    }
    updatePreview();
  });

  // Ecouteurs sur les autres champs texte
  [objectif, contexte, attentes, exemples].forEach(el => {
    el.addEventListener('input', updatePreview);
  });

  updatePreview();
})();
</script>

## Cas pratique — faire évoluer un prompt selon le besoin

**Contexte** : recruter un assistant administratif, rédiger une offre d'emploi attractive.

1. **Étape 1** : écrire un prompt ROFT (Rôle, Objectif, Format, Ton)
2. **Étape 2** : améliorer avec CROFT — ajouter type d'entreprise, ambiance, valeurs de l'équipe
3. **Étape 3** : gérer une situation complexe avec SOCRATE — peu de candidatures, salaire sous le marché, avantages à mettre en avant

## L'IA pour améliorer ses prompts

Demander à l'IA d'améliorer votre propre prompt initial :

!!! example "Prompt d'amélioration"
    *Je viens de te donner ce prompt : [copie ton prompt]. Comment je pourrais l'améliorer pour obtenir une meilleure réponse ? Donne-moi une version améliorée avec explications.*

**Utilité** : itérer rapidement, comprendre les erreurs, gagner en clarté. C'est un processus, pas une solution miracle.

## Reverse prompting — analyser comme un détective

Cas d'usage : analyser un prompt réussi pour comprendre ce qui marche.

**Entrée** : une bonne réponse de l'IA (texte ou image)
**Processus** : se demander « Quel prompt a probablement généré ceci ? » → déduire Rôle, Objectif, Format, Ton
**Sortie** : prompt déduit + suggestions pour l'améliorer
