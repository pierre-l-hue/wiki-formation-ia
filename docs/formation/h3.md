# H3 — Structurer un prompt rédactionnel efficace

Cette troisième séance est consacrée à la maîtrise des méthodes de structuration de prompts textuels (ROFT, CROFT, FAI, SOCRATE), au co-guidage avec l'IA et au reverse prompting pour préparer l'Exercice 2 de la certification[cite: 1].

## Part 1 : Les frameworks de structuration de prompts

### 1. Rappel : La méthode ROFT (Base simple et efficace)

Le framework **ROFT** constitue le socle fondamental pour éviter les réponses génériques[cite: 1] :

* **R — Rôle :** Quelle posture ou expertise l'IA doit-elle adopter ?[cite: 1] *(ex: Expert en communication B2B)*
* **O — Objectif :** Quelle est la tâche exacte à accomplir ?[cite: 1] *(ex: Rédiger un email de relance prospect)*
* **F — Format :** Quelle structure, longueur ou organisation visuelle appliquer ?[cite: 1] *(ex: 2 paragraphes, liste à puces, 150 mots max)*
* **T — Ton :** Quel style, énergie ou niveau de langage employer (et quelles formules éviter) ?[cite: 1] *(ex: Chaleureux mais ferme, sans jargon)*

---

### 2. CROFT : La version complète avec Contexte

Ajouter du **Contexte (C)** avant le bloc ROFT permet d'ancrer l'IA dans votre réalité métier[cite: 1].

> **C — Contexte :** Expliquer la situation en 2-3 phrases (qui vous êtes, le secteur, les enjeux, le destinataire final)[cite: 1].

| Méthode | Exemple de prompt | Résultat obtenu |
| :--- | :--- | :--- |
| **ROFT** | *"Agis comme un juriste. Rédige une lettre de résiliation FAI avec éléments juridiques en 2 parties."*[cite: 1] | Lettre générique adaptable mais neutre[cite: 1]. |
| **CROFT** | *"Je suis client chez Free. Je souhaite résilier mon abonnement box sans engagement au 1er février. Je veux une réponse rapide du service client. Agis comme un juriste. Rédige une lettre de résiliation..."*[cite: 1] | Courrier ultra-ciblé, directement exploitable sans retouche[cite: 1]. |

---

### 3. FAI : Quand la rapidité prime (En 2 minutes)

Pour un besoin express, un brainstorming ou un court message, la méthode **FAI** va droit au but[cite: 1] :

* **F — Formule :** La phrase simple qui résume la demande[cite: 1] *(ex: "Je cherche un nom d'entreprise de paysagisme")*.
* **A — Action :** Ce que doit produire l'IA[cite: 1] *(ex: "Propose 15 noms sous forme de liste numérotée avec une explication pour chacun")*.
* **I — Intention :** L'usage final et le public visé[cite: 1] *(ex: "Noms modernes et mémorables pour des particuliers propriétaires de maison")*.

---

### 4. SOCRATE : Pour les situations complexes & blocages

Lorsque le problème comporte des contraintes lourdes, des tensions ou des paramètres multiples, la méthode **SOCRATE** offre un cadrage chirurgical[cite: 1] :

* **S — Situation :** Quel est le problème initial ?[cite: 1] *(ex: Un fournisseur a du retard sur un paiement)*.
* **O — Objectif :** Quel résultat précis est attendu ?[cite: 1] *(ex: Rédiger un email de relance)*.
* **C — Contexte :** Quels sont les détails clés ?[cite: 1] *(ex: Je suis responsable logistique en PME, le fournisseur est habituellement fiable)*.
* **R — Rôle :** Quelle posture adopter ?[cite: 1] *(ex: Responsable achats expérimenté)*.
* **A — Attentes :** Structure, longueur, éléments indispensables[cite: 1] *(ex: 3 paragraphes, rappel de commande, impact trésorerie, délai)*.
* **T — Ton :** Style et consignes d'attitude[cite: 1] *(ex: Courtois mais ferme, professionnel)*.
* **E — Exemples / Variantes :** Exigence d'illustrations ou d'options[cite: 1] *(ex: Propose 2 variantes de ce mail)*.

## Part 2 : Améliorer ses prompts avec l'IA

### 1. Le méta-prompting (Demander à l'IA d'optimiser le prompt)

Vous n'avez pas besoin de concevoir le prompt parfait du premier coup[cite: 1]. Vous pouvez utiliser le LLM comme un coach en prompting[cite: 1] :

<div class="good-reflex-box">
  <h3>Script d'optimisation de prompt</h3>
  <p><strong>Étape 1 :</strong> Envoyez votre prompt initial simple[cite: 1].<br>
  <strong>Étape 2 :</strong> Relancez immédiatement avec la consigne suivante :</p>
  <blockquote>
    « Je viens de te donner ce prompt : [Copier votre prompt initial]. Comment pourrais-je l'améliorer pour obtenir une réponse encore plus précise et professionnelle ? Donne-moi une version améliorée selon le framework CROFT et explique tes modifications. »[cite: 1]
  </blockquote>
</div>

Cette approche permet de **comprendre les oublis** (manque de ton, format flou), de **gagner en clarté** et d'instaurer une **boucle d'amélioration continue**[cite: 1].

---

### 2. Le Reverse Prompting (Rétro-ingénierie)

Le *Reverse Prompting* consiste à jouer les détectives : vous fournissez à l'IA un texte particulièrement réussi et vous lui demandez de **déduire le prompt exact** qui a permis de le générer[cite: 1].

    🤖 L'IA analyse<br><span style="font-size: 11px; font-weight: normal; color: #666;">(Rôle, Ton, Format)</span>
  </div>

* **Cas d'usage :** Analyser un excellent post de concurrent, une lettre type parfaite ou une note d'analyse pertinente pour en extraire la "recette" sous forme de prompt réutilisable[cite: 1].

(function() {
  const questions = [
    {
      task: "Vous devez générer rapidement 10 idées de sujets de réunion d'équipe en 2 minutes. Quelle méthode est la plus appropriée ?",
      options: [
        "SOCRATE",
        "FAI (Formule, Action, Intention)",
        "CROFT avec un paragraphe d'historique"
      ],
      correct: 1,
      explain: "Pour un besoin ultra-rapide et direct sans contrainte lourde, la méthode FAI est idéale pour aller à l'essentiel en 2 minutes."
    },
    {
      task: "Dans la méthode CROFT, qu'apporte la lettre 'C' par rapport au simple ROFT ?",
      options: [
        "La correction automatique des fautes d'orthographe",
        "Le Contexte (situation, entreprise, secteur) pour éviter les réponses génériques",
        "La contrainte de complexité algorithmique"
      ],
      correct: 1,
      explain: "Le Contexte explique la situation métier précise, ce qui permet à l'IA d'adapter le vocabulaire et les arguments au cas réel."
    },
    {
      task: "Vous devez gérer un litige complexe avec un fournisseur en retard de paiement. Quelle méthode privilégier ?",
      options: [
        "SOCRATE (Situation, Objectif, Contexte, Rôle, Attentes, Ton, Exemples)",
        "FAI",
        "Un prompt d'un seul mot"
      ],
      correct: 0,
      explain: "SOCRATE permet de cadrer les situations sensibles ou conflictuelles en précisant le problème, le ton, l'enjeu et les contraintes."
    },
    {
      task: "En quoi consiste le 'Reverse Prompting' ?",
      options: [
        "À écrire un prompt à l'envers de droite à gauche",
        "À soumettre un bon texte à l'IA pour qu'elle déduise le prompt qui a permis de le créer",
        "À refuser les réponses fournies par l'IA"
      ],
      correct: 1,
      explain: "Le Reverse Prompting fait faire de la rétro-ingénierie à l'IA pour retrouver la structure de prompt d'un excellent livrable."
    }
  ];

  let idx = 0;
  let score = 0;

  const progress = document.getElementById('quiz-progress-h3');
  const taskEl = document.getElementById('quiz-task-h3');
  const choicesEl = document.getElementById('quiz-choices-h3');
  const feedback = document.getElementById('quiz-feedback-h3');
  const nextBtn = document.getElementById('quiz-next-h3');
  const resetBtn = document.getElementById('quiz-reset-h3');
  const scoreEl = document.getElementById('quiz-score-h3');

  function render() {
    if (!taskEl) return;
    feedback.style.display = 'none';
    nextBtn.style.display = 'none';
    resetBtn.style.display = 'none';
    scoreEl.style.display = 'none';

    progress.textContent = "Question " + (idx + 1) + " sur " + questions.length;
    taskEl.textContent = questions[idx].task;
    choicesEl.innerHTML = '';

    questions[idx].options.forEach((opt, oIdx) => {
      const btn = document.createElement('button');
      btn.style.cssText = "text-align:left; padding:10px 14px; border:1px solid var(--md-default-fg-color--lightest, #d5d5d5); border-radius:6px; background:var(--md-default-bg-color, #fff); color:var(--md-typeset-color, #1a1a1a); cursor:pointer; font-size:14px; width:100%; transition: background 0.2s;";
      btn.textContent = opt;
      btn.addEventListener('click', () => answer(oIdx, btn));
      choicesEl.appendChild(btn);
    });
  }

  function answer(oIdx, btn) {
    const q = questions[idx];
    const allBtns = choicesEl.querySelectorAll('button');
    allBtns.forEach(b => b.disabled = true);

    if (oIdx === q.correct) {
      btn.style.background = "rgba(74, 155, 94, 0.2)";
      btn.style.borderColor = "#4a9b5e";
      feedback.style.background = "rgba(74, 155, 94, 0.1)";
      feedback.style.border = "1px solid #4a9b5e";
      feedback.textContent = "✅ Exact ! " + q.explain;
      score++;
    } else {
      btn.style.background = "rgba(201, 86, 74, 0.2)";
      btn.style.borderColor = "#c9564a";
      allBtns[q.correct].style.background = "rgba(74, 155, 94, 0.2)";
      allBtns[q.correct].style.borderColor = "#4a9b5e";
      feedback.style.background = "rgba(201, 86, 74, 0.1)";
      feedback.style.border = "1px solid #c9564a";
      feedback.textContent = "❌ Pas tout à fait. " + q.explain;
    }
    feedback.style.display = 'block';

    if (idx < questions.length - 1) {
      nextBtn.style.display = 'inline-block';
    } else {
      scoreEl.style.display = 'block';
      scoreEl.textContent = "Résultat final : " + score + " / " + questions.length;
      resetBtn.style.display = 'inline-block';
    }
  }

  if (nextBtn) {
    nextBtn.addEventListener('click', () => { idx++; render(); });
  }

  if (resetBtn) {
    resetBtn.addEventListener('click', () => {
      idx = 0;
      score = 0;
      render();
    });
  }

  render();
})();
