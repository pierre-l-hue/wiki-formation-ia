# H2 — Stratégie d'implémentation IA

*Analyser des tâches métier, cartographier les besoins et proposer des outils IA adaptés.*

---

## Objectifs de cette séance

- **Construire une stratégie d'implémentation IA** : analyser un poste, identifier les tâches à forte valeur ajoutée, associer les bons outils.
- **Découvrir des outils spécialisés essentiels** : NotebookLM, Gems (Custom GPTs), Napkin.ai, Remove.bg et Consensus.
- **Modéliser une proposition** : lister les activités, justifier le choix des solutions et estimer le budget (Free vs Pro).
- **S'entraîner sur un cas concret** (Poste d'assistant administratif et commercial).

---

## L'épreuve de certification : Ce qui est attendu

Dans le cadre de la **certification RS6776**, vous devez être capable d'auditer une activité professionnelle et de formaliser un plan d'action d'intégration de l'IA :

1. **Contexte & Enjeux** — Présenter brièvement l'entreprise et le poste ciblé.
2. **Cartographie des tâches** — Lister les missions récurrentes et chronophages.
3. **Périmètre d'automatisation/assistance** — Identifier la valeur ajoutée de l'IA sur chaque tâche.
4. **Sélection des outils** — Proposer l'outil adéquat (Généraliste vs Spécialisé).
5. **Analyse coûts / bénéfices** — Justifier les choix et établir une estimation budgétaire.

---

## Cas pratique d'entraînement : L'Assistant Administratif

!!! info "Profil d'étude"
    **Entreprise :** PME spécialisée dans l'événementiel B2B (organisation de séminaires).  
    **Poste :** Assistant administratif et commercial.  
    **Missions :** Gestion des mails clients, mise en forme de présentations, comptes-rendus de réunion, animation LinkedIn, préparation de la newsletter.  
    **Problème :** Surcharge de sollicitations, tâches répétitives à faible valeur ajoutée, manque de régularité dans la communication.

---

## La boîte à outils de la séance

!!! tip "La règle d'or d'implémentation"
    Commencez toujours par exploiter les capacités d'un **LLM généraliste** (ChatGPT, Gemini, Claude, Mistral) qui couvre 80 % des besoins. N'ajoutez un outil spécialisé que si la tâche exige une fonctionnalité métier spécifique (analyse documentaire stricte, détourage, génération de schémas).

### 1. NotebookLM (Google)
- **Principe :** Un assistant qui raisonne **exclusivement sur vos propres documents** (PDF, Docs, Audio, Sites web). Aucun risque d'hallucination basée sur le web externe.
- **Usage clé :** Synthèse de gros dossiers, création de FAQ, interrogation de cahiers des charges.
- **Tarif :** Gratuit.

### 2. Gems / GPTs personnalisés
- **Principe :** Des assistants IA configurés sur-mesure (rôle, ton, consignes, contraintes) réutilisables à l'infini.
- **Usage clé :** Automatiser la rédaction de tâches récurrentes (CR de réunion, réponses types, posts RS).
- **Tarif :** Gratuit (Gems) ou Inclus dans ChatGPT Plus (GPTs).

### 3. Les outils complémentaires
- 🎨 **Napkin.ai** — Transforme automatiquement du texte ou des listes en schémas visuels (Mindmaps, timelines, process).
- 🖼️ **Remove.bg** — Détourage instantané d'images et photos produits.
- 🔬 **Consensus** — Moteur de recherche scientifique basé sur des études vérifiées et évaluées par des pairs.

(function() {
  const questions = [
    { task: "Répondre à un email client classique demandant un devis.", options: ["IA généralist (ChatGPT / Le Chat)", "NotebookLM", "Napkin.ai", "Consensus"], correct: 0, explain: "Une tâche rédactionnelle courante : un LLM généraliste traite cela rapidement." },
    { task: "Analyser et croiser 15 documents PDF de procédures internes.", options: ["IA généraliste", "NotebookLM", "Remove.bg", "Consensus"], correct: 1, explain: "NotebookLM permet d'interroger vos documents de manière étanche et sans hallucination." },
    { task: "Rédiger chaque semaine un post LinkedIn en respectant une charte éditoriale stricte.", options: ["IA généraliste basique", "Gem / GPT personnalisé", "Napkin.ai", "Consensus"], correct: 1, explain: "Un Gem/GPT mémorise le rôle, le ton et le format pour garantir la cohérence." },
    { task: "Convertir les 5 étapes d'un projet d'événement en schéma visuel pour un client.", options: ["NotebookLM", "Napkin.ai", "Remove.bg", "Consensus"], correct: 1, explain: "Napkin.ai est spécialisé dans la génération automatique de schémas à partir de texte." },
    { task: "Extraire le sujet d'un produit d'une photo pour l'intégrer sur un fond neutre.", options: ["Remove.bg", "NotebookLM", "Gems", "Napkin.ai"], correct: 0, explain: "Remove.bg est l'outil spécialisé idéal pour le détourage rapide." }
  ];

  let i = 0;
  let scoreCount = 0;
  const progress = document.getElementById('quiz-progress');
  const taskEl = document.getElementById('quiz-task');
  const choicesEl = document.getElementById('quiz-choices');
  const feedback = document.getElementById('quiz-feedback');
  const nextBtn = document.getElementById('quiz-next');
  const resetBtn = document.getElementById('quiz-reset');
  const scoreEl = document.getElementById('quiz-score');

  function render() {
    if (!taskEl) return;
    feedback.style.display = 'none';
    nextBtn.style.display = 'none';
    resetBtn.style.display = 'none';
    scoreEl.style.display = 'none';

    progress.textContent = "Question " + (i + 1) + " sur " + questions.length;
    taskEl.textContent = questions[i].task;
    choicesEl.innerHTML = '';

    questions[i].options.forEach((opt, idx) => {
      const btn = document.createElement('button');
      btn.style.cssText = "text-align:left; padding:10px 14px; border:1px solid var(--md-default-fg-color--lightest, #d5d5d5); border-radius:6px; background:var(--md-default-bg-color, #fff); color:var(--md-typeset-color, #1a1a1a); cursor:pointer; font-size:14px; width:100%; transition: background 0.2s;";
      btn.textContent = opt;
      btn.addEventListener('click', () => answer(idx, btn));
      choicesEl.appendChild(btn);
    });
  }

  function answer(idx, btn) {
    const q = questions[i];
    const allBtns = choicesEl.querySelectorAll('button');
    allBtns.forEach(b => b.disabled = true);

    if (idx === q.correct) {
      btn.style.background = "rgba(74, 155, 94, 0.2)";
      btn.style.borderColor = "#4a9b5e";
      feedback.style.background = "rgba(74, 155, 94, 0.1)";
      feedback.style.border = "1px solid #4a9b5e";
      feedback.textContent = "✅ Exact ! " + q.explain;
      scoreCount++;
    } else {
      btn.style.background = "rgba(201, 86, 74, 0.2)";
      btn.style.borderColor = "#c9564a";
      allBtns[q.correct].style.background = "rgba(74, 155, 94, 0.2)";
      allBtns[q.correct].style.borderColor = "#4a9b5e";
      feedback.style.background = "rgba(201, 86, 74, 0.1)";
      feedback.style.border = "1px solid #c9564a";
      feedback.textContent = "❌ Pas tout à fait. " + q.explain;
    }
    feedback.style.display = 'block';

    if (i < questions.length - 1) {
      nextBtn.style.display = 'inline-block';
    } else {
      scoreEl.style.display = 'block';
      scoreEl.textContent = "Résultat final : " + scoreCount + " / " + questions.length;
      resetBtn.style.display = 'inline-block';
    }
  }

  if (nextBtn) {
    nextBtn.addEventListener('click', () => { i++; render(); });
  }

  if (resetBtn) {
    resetBtn.addEventListener('click', () => {
      i = 0;
      scoreCount = 0;
      render();
    });
  }

  render();
})();
