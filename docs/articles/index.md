# Articles

Cet espace est fait pour grandir. Contrairement aux séances H0-H7 (qui suivent un ordre fixe), les articles peuvent être ajoutés dans n'importe quel ordre, à votre rythme.

## Comment ajouter un article

1. Créez un nouveau fichier dans `docs/articles/`, par exemple `docs/articles/mon-nouvel-article.md`
2. Écrivez votre contenu en Markdown (les mêmes règles simples que dans les autres pages : `#` pour un titre, `-` pour une liste)
3. Ajoutez une ligne dans `mkdocs.yml`, sous la section `Articles`, pour qu'il apparaisse dans le menu
4. Enregistrez et publiez (voir le guide de publication) — le site se met à jour automatiquement

!!! tip "Astuce"
    Pas besoin de toucher aux autres pages. Un nouvel article n'a aucun impact sur H0-H7, les fiches outils ou le glossaire.
