    Sources officielles¶

    🎯 Pourquoi cette page est importante¶

    Les outils IA, les règles de confidentialité et les obligations réglementaires évoluent rapidement.

    Avant une session de formation ou une mise à jour du wiki, il est recommandé de vérifier les sources ci-dessous.

    Certification RS6776¶

    • France Compétences — Fiche RS6776

    Cette fiche officielle présente la certification, ses objectifs, ses compétences évaluées et ses modalités.

    Données personnelles et RGPD¶

    • CNIL — Intelligence artificielle
    • CNIL — Comment déployer une IA générative ?

    Ces pages aident à comprendre les précautions à prendre avec les données personnelles et l’usage de l’IA générative.

    IA Act européen¶

    • EUR-Lex — Règlement européen 2024/1689 sur l’intelligence artificielle
    • Commission européenne — Cadre réglementaire sur l’IA

    Ces sources permettent de vérifier les règles officielles liées à l’IA Act.

    Politiques des outils IA¶

    Les politiques de confidentialité des outils IA peuvent changer.

    Avant de recommander un outil pour un usage professionnel, vérifier notamment :

    • si les données sont utilisées pour entraîner les modèles ;
    • si l’historique peut être désactivé ;
    • si les données sont stockées en Europe ou ailleurs ;
    • s’il existe une offre professionnelle ou entreprise ;
    • quelles garanties sont prévues pour les données sensibles.
